#pom-tom

A pomodoro time management tool powered by Electron.

https://en.wikipedia.org/wiki/Pomodoro_Technique
## Instructions

- run `npm install`
- run `npm run build` to make Example.app
- run `npm start` to run app from CLI without building Example.app
